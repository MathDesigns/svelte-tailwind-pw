## Run Locally

Clone the project

```bash
  git clone https://github.com/MathDesigns/svelte-tailwind-pw
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Build it

```bash
  npm run build
```

Run the build

```bash
  node build
```
